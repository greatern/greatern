### 🔐 [ThreatVision](https://github.com/greatern/ThreatVision) — Cybersecurity Monitoring Platform
> `Python` `FastAPI` `MySQL` `SQLAlchemy` `Linux` `REST APIs`

Python-based log analysis engine that detects brute-force login attacks from authentication logs using configurable detection rules. Includes a FastAPI backend for alert retrieval and a normalised relational database for persistent threat storage.

---

### 🗺️ [Route Optimisation Engine](https://github.com/greatern/route-optimizer) — Graph-Based Logistics Pathfinding
> `Java` `Dijkstra's Algorithm` `A* Algorithm` `Graph Theory`

Implements Dijkstra's and A* pathfinding algorithms across weighted graph networks for minimum-cost delivery route computation. Benchmarked against varying graph sizes with documented performance trade-offs.
